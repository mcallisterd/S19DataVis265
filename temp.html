Trains

On the right is a railroad diagram for a list in HTML. The general idea is like any element in HTML, with an opening tag, which may contain attributes, some content, and a closing tag. Here the opening tag is either of the form <ol or <ul and any attributes, maybe something like id="list_1," so that you could link to the top of the list with an <a href="#link"> style tag. Then the tag may just close, although with either an ordered or unordered list, this would result in no visible effect from the code. Then you can place all kinds of HTML elements: images, forms, paragraphs, headings, or list items. List items open with a <li tag, with attributes, perhaps, and then the content for that bullet point or number, if the list is unordered or ordered, respectively. Then the </li> tag to close the item, and repeat as desired. The list as a whole is closed with an </ul> (or ol if the list was ordered). An example follows.
